#+title: Install
#+options: toc:2 num:t

* Requirements

- A Fortran 2018 compiler (gfortran 13+ is what CI and =pixi= pin)
- [[https://fpm.fortran-lang.org][fpm]] for the library and tests
- [[https://fypp.readthedocs.io][fypp]] to regenerate =capnp_endian= / =capnp_message= from =.fypp= sources
- The Cap'n Proto =capnp= tool when compiling schemas or regenerating fixtures

Optional interop tier:

- meson, ninja, a C/C++ toolchain, cmocka
- libcapnp / libcapnp-rpc for the live C++ RPC peer test

* fpm

#+begin_src console
$ git clone https://github.com/HaoZeke/capnp-fortran.git
$ cd capnp-fortran
$ fpm build
$ fpm test
#+end_src

The package name in =fpm.toml= is =capnp=; the executable is =capnpc-fortran=.

* pixi (recommended)

=pixi.toml= pins gfortran, fpm, fypp, and capnproto on =linux-64=:

#+begin_src console
$ pixi install
$ pixi run gen      # fypp → src/*.f90 (commit if templates change)
$ pixi run build
$ pixi run test
#+end_src

Environments:

| Feature | Purpose |
|---------+---------|
| default | library + tests |
| interop | meson + cmocka + C++ for golden master / C++ peer |
| docs | Sphinx + emacs ox-rst export |

* Interop suite

#+begin_src console
$ git clone --depth 1 https://github.com/opensourcerouting/c-capnproto third_party/c-capnproto
$ pixi run -e interop meson setup build-interop .
$ pixi run -e interop meson compile -C build-interop
$ pixi run -e interop meson test -C build-interop -v
#+end_src

#+begin_export rst
When libcapnp-rpc is available, meson also builds the Fortran RPC client
against a C++ ``EzRpcServer`` (schema ``adder.capnp``). Details: :doc:`interop`.
#+end_export

* Building documentation

#+begin_src console
$ pixi run -e docs docs
# HTML under docs/build/
#+end_src

Live site: https://capnp-fortran.rgoswami.me (Cloudflare Pages via the Documentation workflow).

* Using the library from another fpm project

Add a dependency on this repository (or the fpm registry package =capnp= once published), then:

#+begin_src fortran
use capnp
#+end_src

Generated schema modules are ordinary Fortran sources you compile alongside your program after running =capnpc-fortran=.

* CMake / FetchContent

Top-level =CMakeLists.txt= builds the same checked-in =src/*.f90= set as the meson interop tier and exports the =capnp::capnp= target (Fortran =.mod= files on the target include path). Optional plugin: =CAPNP_BUILD_PLUGIN= (default ON when this tree is the top-level project, OFF under =FetchContent= / =add_subdirectory=).

#+begin_src cmake
include(FetchContent)
FetchContent_Declare(
  capnp
  GIT_REPOSITORY https://github.com/HaoZeke/capnp-fortran.git
  GIT_TAG        v0.1.0
)
# Optional: set(CAPNP_BUILD_PLUGIN ON CACHE BOOL "" FORCE)
FetchContent_MakeAvailable(capnp)
target_link_libraries(myapp PRIVATE capnp::capnp)
#+end_src

Standalone:

#+begin_src console
$ cmake -S . -B build-cmake -DCMAKE_BUILD_TYPE=Release
$ cmake --build build-cmake
# library: build-cmake/libcapnp.a  (or .so if CAPNP_BUILD_SHARED=ON)
# plugin:  build-cmake/capnpc-fortran   when CAPNP_BUILD_PLUGIN=ON
#+end_src

Installable package config (=find_package(capnp)=) is generated when =CAPNP_INSTALL= is ON (default for top-level builds).
